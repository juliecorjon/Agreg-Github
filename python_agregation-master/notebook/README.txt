Simulation pour l'agrégation
============================

Installation
------------

En plus des package usuels (numpy, matplotlib, jupyter), il faut installer et activer le package [jupyter matplotlib](https://github.com/matplotlib/jupyter-matplotlib). 

    conda install -c conda-forge ipympl
    conda install -c conda-forge widgetsnbextension


Utilisation 
-----------

Exécuter toutes les cellules du notebook. Il est possible soit de tracer un graph statique (en enlevant les commentaires dans la cellule) soit de lancer l'application interactive.
